Mes Projets

Dans le cadre de mon bootcamp Analyse de données, voici les quatre projets que j'ai réalisés :

Projet 1: Primero Bank - Analyse des Départs de Clients

Contexte du Projet: Primero Bank, une banque 100% en ligne, a observé une hausse de son taux d'attrition et souhaitait comprendre les raisons des départs. L'analyse a utilisé des données démographiques et financières.

Objectifs:

  • Analyser les caractéristiques des clients ayant quitté la banque.
  • Identifier les facteurs influençant les départs.
  • Repérer les segments à risque et proposer des recommandations.

Méthodologie: Nettoyage des données, TCD avec Excel, visualisations dynamiques via Power BI, SQL pour la gestion des données.

Résultats Clés: Taux d'attrition global de 16,15%, taux élevés pour les cartes Blue et Platinum, clients à revenu $60K-$80K et faible utilisation plus susceptibles de partir.

Recommandations: Amélioration des offres pour segments à risque, programmes de fidélisation, flexibilité des options de paiement et formation du personnel.

Vous pouvez consulter le projet (documents et résultats) sur ma page GitHub

Projet 2: Dat’Assur Habitation - Standardiser les tarifs pour offrir des prix compétitifs

Contexte : Analyse du portefeuille de contrats de Dat’Assur pour standardiser la politique tarifaire.

Objectifs :

  • Créer une base de données unifiée.
  • Analyser le portefeuille pour identifier les différences de tarification
  • Standardiser la politique tarifaire
  • Créer des rapports et présentations pour exposer les résultats et la méthodologie

Résultats clés :

  • Prix moyen de la cotisation mensuelle : 19,33 euros.
  • Top 3 des départements par cotisation :
    • Paris : 36,4 euros
    • Hauts-de-Seine : 26,25 euros
    • Val-de-Marne : 19,82 euros
  • Top 3 nombre de contrat par région :
    • Île-de-France : 14 179
    • Provence-Alpes-Côte d’Azur : 3 279
    • Auvergne-Rhône-Alpes : 3 056

Vous pouvez consulter le projet (documents et résultats) sur ma page GitHub
